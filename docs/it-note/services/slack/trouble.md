Message Formatについて
----------------------

### 文字列にリンクURLを埋め込みたい

```text
<https://example.com|Overlook Hotel>
```


リマインドについて
------------------

### 複数行のメッセージをリマインドしたい

ダブルクォーテーションを使う。

```
/remind #your_channel "
* hoge
* hoge
* hoge
" in 10 seconds
```
