接続コマンド
------------

| オプション | 意味           |
| ---------- | -------------- |
| -d         | データベース   |
| -h         | ホスト名       |
| -p         | ポート番号     |
| -U         | ユーザ名       |
| -w         | ノーパスワード |


内部実行コマンド
----------------

| コマンド     | 説明                 | 覚え方                    |
| ------------ | -------------------- | ------------------------- |
| `\l`         | データベース一覧表示 | list                      |
| `\c`         | データベース選択     | connect (choice / change) |
| `\dt`        | テーブル一覧表示     |                           |
| `\d <table>` | テーブル情報表示     |                           |
