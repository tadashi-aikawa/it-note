# [Bookmark Sidebar] Config

{{ page.meta.description }}


⌨ キーバインド
---------------

`Alt + w`でサイドバーを開く。


⚙ 設定
-------

```yaml
サイドバー:
    一般:
        位置: 左
        ブックマークを開く: 現在のタブ
        ページのスクロールを防ぎます: false
        記録する: スクロールの位置と開かれたフォルダ(サブディレクトリの開閉状態を記憶する)
        新規タブ:
            - フォアグラウンドで開く
            - 現在のタブの後
        現在のフォルダをトップに表示します: true
        スクロールバーを隠す: 1.5s
            
    トグル:
        サイドバーを開く:
            トリガー: 左クリック
            エリアの設定:
                幅: 10px
                幅(ウィンドウ表示): 20px
                高さ: 100%
        サイドバーを開く(ブックマーク開いた後): false
        開く時間を遅らせます: 0x
        マウスアウトで閉じる時間:
            時間: 1s
            自動的に閉じない: false
        ウィンドウモードでサイドバーを無効にする: false
    ツールチップ:
        ツールチップコンテンツ:
            内容: タイトルとURL
            追加情報: true
        ツールチップの遅延:
            時間: 0.5s
            ツールチップを表示しない: false
    フィルタ:
        サイドバーを読み込む: すべてのページ
外観:
    一般:
        テーマ: 初期テーマ
        書体: SauceCodePro NF
        配色: default
        マウスオーバーカラー: default
        表面: dark
        テキストの色: default
    サイドバー:
        幅: 400px
        ヘッダの高さ: 50px
        マスク色: default
        フォルダアイコン: default
        フォルダアイコンの色: default
        フォルダの矢印: false
        ブックマークフォントサイズ: 14px
        ブックマーク行の高さ: 38px
        ブックマークアイコンサイズ: 16px
        フォルダアイコンのサイズ: 0px
        フォルダインデント: 25px
        ブックマーク水平パディング: 16px
        スクロールバーの幅: 11px
        ツールチップフォントサイズ: 9px
    インジケータ:
        幅: 24px
        色: default
        アイコンサイズ: 24px
        アイコン色: default
    オーバーレイ:
        マスク色: default
        ヘッダーの高さ: 50px
    アイコン:
        アイコンの形: default
    色: 自動
新規タブ:
    新しいページを上書きする: true
    自動的にサイドバーを開く: true
    アドレスバーを選択する: true
    ウェブサイトを開く: ""
言語:
    一般:
        言語: Japanese
    翻訳: default
```
