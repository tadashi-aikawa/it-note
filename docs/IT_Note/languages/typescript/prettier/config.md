# [Prettier] Config


よく使う設定
------------

`.prettierrc.yaml`
```yaml
printWidth: 120
tabWidth: 2
useTabs: false
semi: true
singleQuote: true
quoteProps: as-needed
trailingComma: all
bracketSpacing: true
arrowParens: avoid
```

設定はTypeScript Deep Diveに準拠している。
