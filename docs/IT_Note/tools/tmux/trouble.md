# [tmux] FAQ

セッション一覧を表示したい
----------------------

アタッチしていないもの含む。

* ショートカットキーなら`<bind-key> w`
* コマンドなら`tmux list-session`


別のセッションをアタッチしたい
------------------------------

* ショートカットキーなら`セッション一覧を表示したい`で選択->`Enter`
* コマンドなら`tmux attach -t <session_no>`


Window一覧画面から特定のWindowを削除したい
------------------------------------------

1. `t`でタグを付ける
2. `:kill-window`
