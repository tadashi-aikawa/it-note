サブメニューの作り方
--------------------

タイプを`メニュー`にして、オプションの`開く`と`閉じる`で囲めばいける。

{{link("http://tablacus.hatenablog.com/entry/2015/06/04/223939")}}

そこで指定した引数のあとに、選択されたエントリが引数として1つ以上入ってくる。
